菜单即工作流

不是拼功能，而是把小红书家居账号的日常动作拆成连续的生产流程。

📊

看板

展示关键提醒：未出稿、久未发布、草稿积压，先做什么一眼可见。

🖼

图库

导入图片后自动识别内容，支持多选合并生成草稿，素材秒变可写对象。

✍️

笔记

AI 助手内嵌编辑器，支持标题生成、正文改写、续写、标签整理与自动保存。

✨

灵感

灵感梦工厂根据历史数据和人设风格，生成可直接保存的成稿候选。

📈

数据

趋势图 + AI 数据对话，帮助定位哪些标题结构和发布时间更容易起量。

⭐

榜样

录入对标账号并做 AI 风格分析，提炼可学习的语气、结构和选题方向。

👤

账号

固化你的人设语气、禁忌词和标签池，让每次生成更像你，而不是像模板。

⚙️

设置

配置 MiniMax Key 与 AI 快捷操作，让团队创作习惯沉淀为可复用指令。

AI 功能特色（真正提效的部分）

围绕创作卡点设计，不做花架子：从看图理解到结构化输出，再到自动发布。

01 图像理解MiniMax VLM 识图分析
02 语气注入读取账号人设与禁忌词
03 结构生成标题/正文/标签分区输出
04 发布执行Playwright 自动发布
05 数据复盘趋势 + AI 对话优化

🧠 AI 助手面板

  • 支持「生成标题 / 改写开头 / 续写正文」快捷操作
  • 结构化提取候选标题与标签，应用到编辑器更快
  • 生成中可停止，结果可重生，适合迭代调风格

🔮 灵感梦工厂

  • 基于历史内容表现自动给出标题方向
  • 生成稿不污染正式列表，确认后再保存为草稿
  • 适合选题焦虑时快速拉出多版本候选

📉 数据 AI 对话

  • 把真实发布数据变成可对话的创作建议
  • 找出高表现标题长度、标签组合、发布时间窗
  • 让内容优化从拍脑袋变成可解释的策略迭代

技术栈与接入方式

前后端分离 + 桌面端 + Agent 协议，既能手动使用，也能自动化运行。

🧩 三端架构

GUI (Tauri + React)
CLI (app/cli.py)
MCP Server (app/mcp/server.py)
   │
FastAPI (app/server.py:8765)
   │
SQLite (data/app.db) + assets/ + crawler/
